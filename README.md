# 🚀 Bubi - eine klitzekleines Projekt 🚀 #

### 😉 Nein, Bubi ist kein kleiner Junge sondern ein Projektname!

Mangels einer anderen, vernünftigen Idee musste eben __Bubi__ her!
### 🛠️ Inhalt
Ein sauberes, gut strukturiertes Endprodukt mit einem kleinen, überschaubaren __main-File__.<br>
Der Löwenanteil wird im include-File __bubi.hpp__ umgesetzt.<br>
Herzstück ist eine globale Struktur, die über __Referenzen__ den einzelnen Klassen zur Verfügung gestellt wird.<br>
Diese Struktur enthält sowohl die _konstanten_ und die _variablen_ Daten.
